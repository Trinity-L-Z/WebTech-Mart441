The images I chose for this project are scans of old film photographs I have of my mother. I had taken the photos and dragged them along the scanner during scanning to get interesting results, and I've been looking for a way to show them off!

I locally downloaded jQuery and included that in the lib folder. I referenced the homework examples to try and copy the document setup, since I usually set up my files differently. I created an array for the images and animated them fading in and out while they looped through. I then animated a star moving visibly across the page (to satisfy assignment requirements). I created an additional array, "adj" to sift through various word pairings with the consistant "mama".
